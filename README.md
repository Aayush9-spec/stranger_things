## ğŸš€ Getting Started

Follow these steps to explore the Stranger Things universe locally:

### 1. Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### 2. Installation

Navigate to the project directory:

```bash
cd vite-project
```

Install the necessary dependencies:

```bash
npm install
```

### 3. Running the App

Start the local development server:

```bash
npm run dev
```
